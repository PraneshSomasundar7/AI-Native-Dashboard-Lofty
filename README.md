🏢 Lofty: AI-Native Real Estate Command Center
Lofty is a next-generation, high-end Real Estate CRM built to supercharge agent productivity. Designed as an intelligent "Command Center," it bridges the gap between passive property tracking and proactive closing strategies through real-time telemetry and AI-driven automation.

✨ Key Features
🧠 AOS Priority Zone (Agent Operating System): An intelligent AI engine that actively monitors lead temperature (e.g., lead scoring, property views). When a lead indicates high interest, AOS alerts you and auto-generates a personalized SMS to instantly engage the buyer.
⚡ Real-Time Telemetry & Live Activity Feed: Never miss a beat. Track exactly when buyers favorite a property, view listings, or respond to messages in real-time.
📱 Integrated Buyer Portal: A unified ecosystem where buyers can browse rich property cards, accept showing invitations, and communicate directly with their agent seamlessly.
📊 Dynamic Data Components: Custom widgets for Today's Tasks, My Listings, Transactions, and Keep In Touch reminders—all engineered to keep agents focused on revenue-generating activities.
💎 Premium, World-Class UI/UX: Built with a stunning bg-slate-50 aesthetic, deep drop-shadow scaling, micro-interactions, and frictionless Tailwind CSS v4 styling to deliver an enterprise-grade SaaS experience.
🛠️ Technology Stack
Frontend Core: React, Vite
Styling Engine: Tailwind CSS v4 (Custom UI framework with beautiful gradients, glassmorphism, and structural animations)
Icons: Lucide React
AI / Backend Integration: InsForge Realtime DB & AI chat completions processing natural language SMS drafts.
🚀 Getting Started
Prerequisites
Make sure you have Node.js and npm installed.

Installation
Clone the repository

bash
git clone https://github.com/your-username/AI-Native-Dashboard-Lofty.git
cd AI-Native-Dashboard-Lofty/lofty-dashboard
Install dependencies

bash
npm install
Start the development server

bash
npm run dev
View the Dashboard Open your browser and navigate to the localhost port provided (usually http://localhost:5173).
